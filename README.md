# CODSOFT Java Development Internship

This repository contains the Java projects completed as part of my Java Development Internship at CodSoft.

## Tasks Completed

### Task 1: Number Guessing Game
**Description:**
- Generates a random number between 1 and 100.
- Allows the user to guess the number.
- Provides hints whether the guess is too high or too low.
- Limits the number of attempts.
- Supports multiple rounds.
- Displays the user's score.

**Technologies Used:**
- Java
- Scanner Class
- Random Class

---

### Task 2: Student Grade Calculator
**Description:**
- Accepts marks for multiple subjects.
- Calculates total marks.
- Calculates average percentage.
- Assigns grades based on the average percentage.
- Displays the final result.

**Technologies Used:**
- Java
- Loops
- Conditional Statements
- Scanner Class

---

### Task 3: ATM Interface
**Description:**
- Check account balance.
- Deposit money.
- Withdraw money.
- Validates user input.
- Prevents withdrawal when balance is insufficient.
- Uses Object-Oriented Programming concepts.

**Classes Used:**
- ATM
- BankAccount
